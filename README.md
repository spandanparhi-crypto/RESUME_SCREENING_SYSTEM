# 📄 Resume Screening System

An AI-powered **Resume Screening System** that automatically analyzes, compares, and ranks multiple resumes against a given job description. This project helps recruiters and HR professionals save time by automating the resume screening process using intelligent text matching and candidate scoring.

🌐 **Live Demo:** (https://closed-olive-j6odgkop.edgeone.dev/)

---

# 🚀 Features

- 📂 Upload multiple PDF resumes
- 📋 Enter or paste Job Description (JD)
- 🤖 AI-powered resume analysis
- 📊 Resume match percentage calculation
- 🏆 Automatic candidate ranking
- 🔍 Skill extraction from resumes
- 📄 Resume preview
- ⚡ Fast screening process
- 📈 Candidate score breakdown
- 📑 Resume summary generation
- 📥 Download screening report
- 📱 Responsive UI

# ⚙️ How It Works

### Step 1

Upload one or multiple resumes (PDF format).

### Step 2

Enter the Job Description.

### Step 3

Click **Analyze Resumes**.

### Step 4

The system extracts resume content.

### Step 5

Skills and keywords are identified.

### Step 6

Each resume is compared with the Job Description.

### Step 7

A matching score is calculated.

### Step 8

Candidates are ranked automatically.

### Step 9

View detailed candidate reports.

### Step 10

Download the final screening report.
